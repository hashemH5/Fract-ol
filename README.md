# Fract-ol

## Description
Fract-ol is a graphical fractal exploration program. 
It allows the user to visualize the Mandelbrot and Julia sets, 
zoom in/out, and explore the structure of these fractals interactively.

## Instructions
```bash
make

./fractol mandelbrot
./fractol julia
./fractol julia 0.285 0.01

```
